ishiko orbit — Space Typing Mission
A browser-based typing speed test with a space mission theme. Type the words as fast and accurately as you can before the 30-second countdown hits zero.

How to Play

Open challenger-final.html in any browser — no setup, no install
Start typing the words you see on screen
Press Space to submit each word
The timer starts automatically on your first keystroke
When time runs out, your mission debrief shows your score


Controls
KeyActionSpaceSubmit current wordTabRestart immediately

Scoring

WPM — correct words typed per minute, updates live
Accuracy — percentage of words you got right
Mission Successful — 50+ WPM and 90%+ accuracy
Mission Failed — anything below that, retry


Features

Animated starfield background
Live WPM and accuracy tracking
Progress bar that drains as time runs out
Word window scrolls as you type — always shows what's coming next
Input turns red if your current typing doesn't match the word
Results screen with full breakdown
